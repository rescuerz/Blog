    Chapter 1 | Welcome Aboard¶

    1.1 Two recurring themes 两个反复出现的主题¶

    • the notion of abstaction（抽象化的概念）
    • the importance of not separating in you mind the notions of hardware and software （不要将软件与硬件分割开）

    1.1.1 the notion of abstraction¶

    • It will be the central to all that you will learn and expect to use in practicing your craft（技巧）
    • Abstraction is a technique for establishing a simpler way for a person to interact with a system（人与系统交互）, removing the details that are unnecessary（去除不必要的细节） for the person to interact effectively with that system.也就是着眼于高层次，聚焦于关键部分
    • 抽象能力实际上也是一种模型提取能力，将你所关注的内容从一个复杂的系统中提炼出来，一般情况下我们省略的都是这个“内容”的具体实现，视其为一个黑箱，而更加关注如何使用这样一个黑箱、这样一个黑箱在各种情况下有怎样的行为。
    • 然而这并不意味着，不去着眼于细小的部分。对逻辑门的设计，如果逻辑电路工作出现问题，查看门的内部结构并查看其功能是否导致问题通常会有所帮助。先大后小的解决思路。
    • 但如果我们必须将多个组件组合成一个更大的系统，我们应该小心，不要让它们的抽象成为我们理解的最深层次。如果我们不知道其抽象级别以下的组件，那么我们就会受到它们在没有我们干预的情况下一起工作的摆布。如果它们不能一起工作，并且我们无法低于抽象层次，我们就会陷入困境。我们应该注意不要陷入这种状态。

    1.1.2 hardware and software¶

    就软件而言，它们通常指的是程序，无论是 Android、ChromeOS、Linux 或 Windows 等操作系统，还是 Access、MongoDB、Oracle 或 DB-terrific 等数据库系统，还是 Facebook、Chrome、Excel 等应用程序，或字。这意味着这个人对这两件事中的一件事了解很多，而对另一件事知之甚少。通常，进一步的暗示是，成为其中一个（硬件或软件）的专家，而对另一个一无所知是可以的。就好像硬件（计算机及其实际工作方式）和软件（指导计算机执行命令的程序）之间有一堵大墙，人们应该满足于留在一边那堵墙或另一堵墙的。

    总之就是，希望同时掌握硬件和软件知识

    1.2 Computer system¶

    A computer system generally includes, in addition to the processor（CPU）, a keyboard for typing commands（键盘输入命令行）, a mouse or keypad or joystick for positioning on menu entries, a monitor（显示器） for displaying information that the computer system has produced, memory for temporarily storing information（内存短时间存储）, disks and USB memory sticks of one sort or another for storing information for a very long time（硬盘长时间存储）, even after the computer has been turned off, connections to other devices such as a printer for obtaining paper copies of that information, and the collection of programs (the software) that the user wishes to execute.

    电脑的操作系统，不仅仅是CPU，还包括一系列附加组件（键盘，鼠标，显示器，内存，硬盘等等）

    1.3 Two very important ideas¶

    • idea_1：
    • All computers (the biggest and the smallest, the fastest and the slowest, the most expensive and the cheapest) are capable of computing exactly the same things if they are given enough time and enough memory.

    在给定充足的时间和内存情况下，所有的计算机能执行的任务是相同的，快速计算机和慢速计算机的区别在于完成时间不同。

    Some computers can do things faster,but none can do more than any other

    • idea_2:
    • It is necessary to transform our problem from the language of human to the voltages that influence the flow of electrons.

    计算机无法理解人类的语言（自然语言，nature language），需要转化为影响电子流动的电压。所以我们需要用一种计算机能够“执行”（而非“理解”，计算机不需要理解，它只需要执行，所以之后说的东西叫“指令”）的“语言”去指导计算机解决问题。

    1.4 computer as universal computational devices（通用计算设备）¶

    computer的独特之处：

    You can tell a computer how to add numbers. You can tell it how to multiply. You can tell it how to alphabetize a list or perform any computation you like. When you think of a new kind of computation,you do not have to buy or design a new computer. You just give the old computer a new set of instructions (a program) to carry out the new computation.

    1.5 How do we get the electrons to do the work?（我们该如何使电子完成我们的任务）¶

    依次从problem，algorithms，language，ISA，microarchitecture角度进行解答

    1.5.1 problem——filled with ambiguity（模糊性）¶

    Such ambiguity would be unacceptable in instructions provided to a computer. The computer, electronic idiot that it is, can only do as it is told. To tell it to do something where there are multiple interpretations would cause the computer to not know which interpretation to follow.

    由于问题具有模糊性，存在着多种解释，故计算机不知道遵循何种解释

    1.5.2 Algorithm¶

    An algorithm is a step-by-step procedure that is guaranteed to terminate, such that each step is precisely stated and can be carried out by the computer.

    • definiteness:describe the notion that each step is precisely stated.
    • 正确性（没有歧义），每一步都被正确地定义
    • effective computability:describe the notion that each step can be carried out by a computer.
    • 有效可计算性”来描述每个步骤都可以由计算机执行的概念。take the largest prime number,不存在可执行性
    • finiteness: describe the notion that the procedure terminates.
    • 有限性，即程序计算最终会停止

    1.5.3 program¶

    transform the algorithm into a computer program in one of the programming languages that are available. Programming languages are “mechanical languages.”

    编程语言（机器语言）不存在模糊性，是对计算机的指令

    two kinds of programming languages

    • high-level languages（C，C++）

    • High-level languages are at a distance (a high level) from the underlying computer. At their best, they are independent of the computer on which the programs will execute. We say the language is “machine independent.”

    • 高层次语言远离计算机的底层框架，

    • low-level languages（LC-3，MIPS，RISC-V）

    • Low-level languages are tied to the computer on which the programs will execute. There is generally one such low-level language for each computer. That language is called the assembly language for that computer.

    • 汇编语言

    • complier 编译器将高级语言直接转化成ISA

    • Assembler 汇编器将低级语言（汇编语言）转化为ISA

    1.5.4 ISA——instruction set architecture（指令集架构）¶

    The instruction set architecture (ISA) is the complete specification of the interface between programs that have been written and the underlying computer hardwarethat must carry out the work of those programs.

    指令集架构（ISA）是已编写的程序与必须执行这些程序工作的底层计算机硬件之间接口，软件与硬件的接口

    • The term opcode(指令操作码) is used to describe the operation.

    • The term operand(操作数) is used to describe individual data values.

    • The ISA specifies the acceptable representations for operands. They are called data types. A data type is a representation of an operand such that the computer can perform operations on that representation. The ISA specifies the mechanisms that the computer can use to figure out where the operands are located. These mechanisms are called addressing modes(寻址模式).

    the number of opcodes,data types,and addressing modes specified by an ISA vary among different ISAs.

    不同的ISA指定的操作码数量、数据类型、寻址模式各不相同。

    Some ISAs have as few as a half dozen opcodes,whereas others have as many as several hundred. 一些 ISA 的操作码少至六个，而其他 ISA 的操作码则多达数百个。

    Some ISAs have only one data type, while others have more than a dozen.

    Some ISAs have one or two addressing modes, whereas others have more than 20.

    The x86, the ISA used in the PC, has more than 200 opcodes, more than a dozen data types, and more than two dozen addressing modes.

    一些 ISA 的操作码少至六个，而其他 ISA 的操作码则多达数百个。有些 ISA 只有一种数据类型，而另一些则有十几种。有些ISA有一种或两种寻址模式，而另一些则有20多种。PC中使用的x86 ISA有200多种操作码、十多种数据类型和两打以上寻址模式。

    • The translation from a high-level language to the ISA of the computer on which the program will execute is usually done by compiler（编译器）, and the translation from the unique assembly language of a computer to its ISA is done by an assembler.

    high-level language 到 ISA的转化，需要编译器compiler

    1.5.5 microarchitecture,logic circuit,device¶

    • Infinite microarchitectures could exist for a single ISA，无穷多个微架构适用于一个ISA eg:汽车的构造相同，但是类型不同

    • one ISA is normally implemented by a single microarchitecture eg:一个汽车只有一个类型
