    Circuit Optimization

    2.3 Circuit Optimization 电路优化¶

    目标：对于给定的函数，获得最简单的实现方式

    Distinct cost criteria we will use:（判断标准）

    • Literal cost (L)文字成本
    • Gate input cost (G) 门输入成本
    • Gate input cost with NOTs (GN) 带NOT的门输入成本

    2.3.1 Literal Cost（L）¶

    literal is a variable or its complement 字面量——变量或者它的补集

    literal cost is the number of literal appearances in a Boolean expression corresponding to the logic circuit diagram.

    数一数，布尔表达式中的变量数即可（可重复计算）

    • \(F = BD + A\bar BC + A\bar C \bar D\)

    L = 8

    • \(F = BD + A\bar B C + A\bar B \bar D + ABC\)

    L = 11

    2.3.2 Gate Input Cost（G）¶

    Gate input costs is the number of inputs to the gates in the implementation corresponding exactly to the given equation or equations. (G - inverters not counted, GN - inverters counted)

    与给定方程完全对应的电路实现中，门的输入数量

    在L的基础上，在加上项的数量（但是不包括单变量项，否则会与L重复计算）

    什么是项：形如A AND B / A OR B的式子，A，B称之为项

    • all literal appearances(L)
    • the number of terms excluding single literal terms, (G). e.g. for A+BC its L=3 while G=1

    2.3.3 Gate input cost with NOTs (GN)¶

    在G的基础上，加上非门的数量(根据变量来，不重复计算相同变量的非门)

    • \(F = BD + A\bar BC + A\bar C \bar D\)

    L = 8 , G= 8 + 3 =11 , GN = 11 + 3 = 14

    • \(F = BD + A\bar B C + A\bar B \bar D + ABC\)

    L = 11 G = 11 + 4 = 15 , GN = 15 + 2 = 17（仅计算BD的非门）

    2.3.4 Map Manipulation¶

    2.3.4.1 Karnaugh maps(K-map) 卡诺图¶

    The K-map can be viewed as

    • A reorganized version of the truth table 真值表的重组版本
    • A topologically-warped Venn diagram as used to visualize sets in algebra of sets 用于可视化集合代数中的集合的拓扑扭曲维恩图。

    简单来说，K-map是小方块的集合，每一个正方形代表一个最小项，相邻的方块只有一个变量不同

    `Adjacent squares differ in the value of one variable

    • m0 和 m1 的不同之处在于 变量y
    • m0 和 m2 的不同之处在于 变量x

    注意列, **`要按照Gray codes的顺序排列, 即相邻两列只相差一位.** 因此列标号为 00 01 11 10.

    相邻两个小方格都只有一个变量不同.

    可以卷起来, 即最左边一列和最右边一列,(这两列也是相邻)

    Alternative Map Labeling

    Combining Squares

    可以画 2^n^ 大小的方形格(即一次要包括 2^n^ 个小方格) For 3-variable K-Map:

    • One square represents a minterm with three variables
    • Two adjacent squares represent a product term with two variables
    • Four “adjacent” terms represent a product term with one variable
    • Eight “adjacent” terms is the function of all ones (no variables) = 1.

    化简思路: 用最大的最少的圈把所有的 1 圈完.(圈与圈之间可以有交叉)

    注意四个角也可以卷(0 2 8 10 -> \(\overline{XZ}\))

    2.3.4.2 Don’t Cares(无关项) in K-Maps¶

    By placing “don't cares” (an “x” entry) in the function table or map, the cost of the logic circuit may be lowered. 如我们把 BCD 码作为输入, 有效数字 0 - 9, 如果那么 9 以上的输入组合我们不在乎输出结果, 称为无关项, 在卡诺图上画 x.

    用 \(d（\cdots）\) 表示无关项，无关项的值既可以是0，也可以是1，根据题目而定

    Find the optimum POS solution:

    \(F(A,B,C,D) = \sum m(3,9,11,12,13,14,15) + \sum d(1,4,6)\)

    Hint: Use \(\bar F\) and complement it to get the result. ans: F' = B' D' + A' B (求 F' 时在卡诺图中圈空格，而不是 1) F = (B + D)(A + B')

    利用卡诺图化简以下表达式 $$ F = AB + \bar BC + CD + \bar AC $$

    2.3.4.3 Systematic Simplification 系统化简化¶

    • A Prime Implicant(质蕴含项) is a product term obtained by combining the maximum possible number of adjacent squares in the map into a rectangle with the number of squares a power of 2.

    素蕴含项是通过将图中最大可能数量的相邻正方形组合成一个矩形而获得的乘积项，其中正方形的数量为 2 的幂。

    如果已经建成一个大的矩形，大矩形里面的小矩形就不做考虑了

    • A prime implicant is called an Essential Prime Implicant(必要质蕴涵项) if it is the only prime implicant that covers (includes) one or more minterms.

    该质蕴含项是唯一覆盖某小正方形的质蕴含项，则称为必要质蕴含项

    也就是，如果没有 必要质蕴含项，就无法实现全覆盖

    质蕴涵项是对某个 "1" 而言包括它的最大方形。对某个 "1" 而言如果它的质蕴涵项只有一个，那么它是必要的。

    1. Find All Prime Implicants
    2. Find Essential ones
    3. 处理剩下的 1

    2.3.5 Multi-Level Circuit Optimization¶

    两级门的延迟低，但 cost 高。 $$ \begin{aligned} F &= \bar A \bar C \bar D + \bar A B \bar C + ABC + AC\bar D G = 16\ F &= \bar A (\bar C \bar D + B\bar C) + A(BC + C\bar D) G = 18\ F &= \bar A \bar C(B + \bar D) + AC(B + \bar D) G = 12\ F &= (\bar A \bar C + AC)(B + \bar D) G = 10 \end{aligned} $$

    2.3.5.1 Integrated Circuits 集成电路¶

    • Integrated circuit (informally, a “chip”) is a semiconductor crystal (most often silicon) containing the electronic components for the digital gates and storage elements which are interconnected on the chip.

    • Terminology - Levels of chip integration

    • SSI (small-scale integrated) - fewer than 10 gates

    • MSI (medium-scale integrated) - 10 to 100 gates
    • LSI (large-scale integrated) - 100 to thousands of gates

    • VLSI (very large-scale integrated) - thousands to 100s of millions of gates

    • Technology Parameters技术参数

    • Fan-in 扇入系数 – the number of inputs available on a gate 一个门的输入端的上限 理想情况一个门的输入可以接无数个门的输入，但实际上因为每一个门的输入端都会花费一些电流，而门输出的电流是有限的，从集成电路本身会要求一个门的输出最多能拉多少门的输入。

    • Fan-out 扇出系数 – the number of standard loads driven by a gate output 一个门的输出带了很多门的输入？（像火车头带了很多车厢） 一个门的输出不能接到太多门的输入上去，一般会用 buffer 做缓冲

      • eg: 1 standard load equals the load contributed by the input of 1 inverter.

      一个标准负载等于一个非门输入所贡献的负载

      • Transition time 转换时间- the time required for the gate output to change , \(from \ H \ to \ L,t_{HL},or \ from L \ to \ H , t_{LH}\)

      带的门越多，上升时间就越多。 我们会限制过渡时间，设置 maximum fan-out

      • The maximum fan-out that can be driven by a gate is the number of standard loads the gate can drive without exceeding its specified maximum transition time

      最大扇出系数，是指不超过其指定最大转换时间的标准负载数

      • Logic Levels – the signal value ranges for 1 and 0 on the inputs and 1 and 0 on the outputs (see Figure 1-1)

      • Noise Margin – the maximum external noise voltage superimposed on a normal input value that will not cause an undesirable change in the circuit output

        叠加在正常输入值上的最大外部噪声电压，不会导致电路输出发生不良变化

      • Cost for a gate - a measure of the contribution by the gate to the cost of the integrated circuit Ignoring the wiring area, the gate area is roughly proportional to the gate input count.

      • Propagation Delay(传输延迟) – The time required for a change in the value of a signal to propagate from an input to an output

        信号值的变化从输入传播到输出所需的时间

      • Power Dissipation功耗 – the amount of power drawn from the power supply and consumed by the gate

    2.3.5.2 Propagation Delay¶

    Delay is usually measured at the 50% point with respect to the H and L output voltage levels. High-to-low (\(t_{PHL}\)) and low-to-high (\(t_{PLH}\)) output signal changes may have different propagation delays.

    计算中点的时间差，延迟通常是在相对于H和L输出电压电平的50%点测量的

    此处需要注意的是，中间的是从低到高，\(t_{PLH}\)

    Delay Mode

    1. Propagation Delay传输延迟

    • Transport delay - a change in the output in response to a change on the inputs occurs after a fixed specified delay 输出整体往后移（固定的指定延迟）

    • Inertial delay（惯性延迟） - similar to transport delay, except that if the input changes such that the output is to change twice in a time interval less than the rejection time, the output changes do not occur. Models typical electronic circuit behavior, namely, rejects narrow “pulses” on the outputs

    如果输入发生变化，使得输出在小于拒绝时间的时间间隔内发生两次变化，则不会发生输出变化。

    除了输出往后移，在惯性延迟下，很窄的脉冲（小于 rejection time）会被消除掉。

    也就是说，Inertial Delay不仅要考虑传输延迟（transport delay），还要考虑消除窄脉冲

    1. circuit delay 电路延迟

    • 最开始 S 由$ 0->1 $后 \(0.9s Y 从 0->1\)（只需要考虑 第二个AND门 和 最后的OR门，第一个AND门不做变化）

    • S 从 1 变为 0后，经过 0.4s 后，第二个AND门变为0，经过0.9s后，Y值从1变为0

    • 非门需要在0.2s后，才能做出相应，从0变为1，经过0.6s后，第一个AND门变为1，经过1.1s后，Y值变为1；
    • 所以在0.9s 到 1.1s 这段时间，Y值为0，出现一个小凹槽，1.1s之后，Y值变为1
    • 这里 Y 出现了一个小尖峰，我们称之为电路产生的冒险。 S 的两条路径我们发现延迟不同，这种我们称之为电路中的竞争。
    1. Fan-out and Delay The fan-out loading a gate’s output affects the gate’s propagation delay。 SL(Standard Load)标准负载: 以非门为标准。带一个非门需要...的负载

    One realistic equation for tpd for a NAND gate with 4 inputs is: $t_{pd} =0.07+0.021SL $ .

    ns SL is the number of standard loads the gate is driving, i. e., its fan-out in standard loads For SL = 4.5, tpd = 0.165 ns 由工艺程度+负载情况决定

    例题：

    NAND gate G with 20 standard loads on its output has a delay of 0.45 ns and has a normalized cost of 2.0.

    与非门G 输出上有20个标准负载，且传输延迟为0.45ns，标准成本为2.0

    A buffer H has a normalized cost of 1.5. The NAND gate driving the buffer with 20 standard loads gives a total delay of 0.33 ns

    缓冲区H的标准成本是1.5，总延迟是0.33ns

    1. The cost of this portion of the circuit cannot be more than 2.5
    2. The delay of this portion of the circuit cannot be more than 0.40 ns
    3. The delay of this portion of the circuit must be less than 0.40 ns and the cost less than 3.0

    ans:

    1. No buffer

    2. Use buffer

    3. Irrelevant - buffer needed to satisfy delay constraint, but cannot satisfy cost constraint.

    无关紧要 - 需要缓冲来满足延迟约束，但不能满足成本约束。

    2.3.6 Additional Gates and Circuits¶

    2.3.6.1 Buffer¶

    没有逻辑功能（相当于导线），但有很强的带负载的能力，能够拉高电平，降低传输延迟。

    A buffer is an electronic amplifier used to improve circuit voltage levels and increase the speed of circuit operation.缓冲器是一种电子放大器，用于提高电路电压电平并提高电路运行速度。

    2.3.6.2 NAND¶

    与非也可以认为是先非后或 \(F= \bar X + \bar Y + \bar Z\)

    A NAND gate with one input degenerates to an inverter.

    Universal gate（通用门） - a gate type that can implement any Boolean function. 最高效

    我们一般不使用与非门做逻辑运算，因为他不满足交换、结合律。一般在最后把与或非(AND OR NOT)转为用与非(NAND)实现。

    2.3.6.3 NOR¶

    也可以利用 De Morgan 变为 先非后与，其他同与非。

    \(F= \bar X \bar Y \bar Z\)

    2.3.6.4 Exclusive OR / Exclusive NOR¶

    definition

    $$ \begin{aligned} &异或的作用相当于加法，当且仅当X和Y不相等时，X \oplus Y = 1\ &因为在此时，X\overline Y,\overline XY才有可能等于1\ &对于XNOR，当且仅当X和Y相等时， \overline{X\oplus Y} = 1\ &因为，在此时，XY或者\overline{X} \overline Y 才有可能等于1 &XNOR的推导过程：\ &\overline{X \oplus Y} = \overline{X\overline Y + \overline XY} = \overline{X\overline Y} \overline{\overline XY} = (\overline X + Y)(X + \overline Y) = XY + \overline{X} \overline Y \end{aligned} $$

    $$ X \oplus Y = X\bar Y + \bar X Y\ \overline{X \oplus Y} = XY + \bar X \bar Y\ X \oplus \bar Y = X\bar{\bar Y} + \bar X \bar Y = XY + \bar X \bar Y = \overline{X \oplus Y}\ $$ 严格来说，只有两变量的异或。对于多于两个输入的，我们用奇函数/偶函数代替。 对于三变量的异或 \(X\oplus Y = \bar X \bar Y Z + \bar X Y \bar Z + X\bar Y \bar Z + XYZ\) 卡诺图无法再化简

    1 1
    1 1

    我们称它为奇函数，因为 minterm 里 1 的个数(=$=\sum m(1(001),2(010),4(100),7(111))) $为奇数。其反函数为偶函数。

    • The 1s of an odd function correspond to minterms having an index with an odd number of 1s.
    • The 1s of an even function correspond to minterms having an index with an even number of 1s.
    \[ \begin{aligned} &对于奇函数和偶函数的论断，对于三变量的异或，要求具有奇数个1，才能得到X\oplus Y \oplus Z = 1\\ &称为奇函数的原因是， minterm里1的个数为奇数，其反函数称为偶函数（minterm里1的个数为偶数）\\ & X \oplus Y \oplus Z = \sum m_1(001) + m_2(010) + m_4(100) + m_7(111) \end{aligned} \]

    The symbols for XOR and XNOR

    $$ \begin{aligned} &至于如何实现复杂门电路异或门\ &X \oplus Y = X\overline Y + \overline X Y\ &需要两个AND门，每个AND门分别对应一个NOT门，最后连接到OR门\ \end{aligned} $$

    Parity Generators and Checkers 奇偶校验：传输若干个数据位，增加一个数据位，用来表示数据位的奇偶性。输入和输出端的奇偶位应该相同。

    2.3.6.5 Hi-Impedance Outputs 高阻输出¶

    一般情况下，不允许门的输出接在一起

    高阻意味着，回顾电路，输出似乎已断开。

    就好像内部电路和输出之间的开关已打开。

    对于符号和真值表，IN 是数据输入，EN 是控制输入。

    当 EN = 0 时，无论 IN 上的值（用 X 表示）如何，输出值都是 Hi-Z。

    当 EN = 1 时，输出值跟随输入值

    Data Selection Function: If s = 0, OL = IN0, else OL = IN1
