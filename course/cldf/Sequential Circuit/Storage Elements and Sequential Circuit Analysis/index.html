    Storage Elements and Sequential Circuit Analysis

    4.1 Storage Element and Analysis¶

    时序逻辑电路包括：组合逻辑电路 + 存储元件

    Other inputs, State or Present State, are signals from storage elements.

    其他输入（状态或者当前状态），是来自存储元件的信号

    The remaining outputs, Next State are inputs to storage elements

    剩余的输出、下一状态是存储元件的输入

    1. Combinatorial Logic
    • Next state function
    • Next State = f(Inputs, State) 次态方程，输入和当前状态共同决定下一状态
    • Output function (Mealy) Mealy模型
    • Outputs = g(Inputs, State)
    • Output function (Moore) Moore模型
    • Outputs = h(State) 输入不会直接改变输出，而是通过状态来间接改变输出

    Mealy 模型和 Moore 模型

    1. Types of Sequential Circuits 时序电路的类型

    • Depends on the times at which:

    • storage elements observe their inputs, and

    • storage elements change their state

      存储元件观察输入并改变其状态

    • Synchronous 同步

    所有的存储元件更新一定发生在时钟周期的整周期上

    • Asynchronous 异步 存储元件更新可以在任意时间发生 如果时钟也被看做一个输入，那么所有电路都是 Asynchronous Asynchronous 可以让我们在有需要的时候更新电路，降低电路的功耗

    一般来说，异步电路的设计相对困难（行为与门的传播延迟和输入信号变化的时间序列密切相关）

    4.1.1 Latch¶

    开启震荡模式，每1.1ns变化一次，前提是S = 0

    4.1.1.1 Basic （NAND）\(\overline S - \overline R\) Latch¶

    R S Q \(\overline Q\) Comment
    1 1 ? ? Stored state unknown
    1 0 1 0 Set Q = 1
    1 1 1 0 now Q remembers 1
    0 1 0 1 Reset Q = 0
    1 1 0 1 now Q remembers 0
    0 0 1 1 Both go high
    1 1 ？ ？ unstable
    • $S = 0 , R = 0 $ is forbidden as input pattern

    因为Q和\bar Q不再满足取反的条件，当R，S同时变为1时，进入存储状态时，Q和 \bar Q的值需要改变且无法确定

    • 第一步和最后一步，unstable是因为，无法确定Q还是\(\bar Q\)变成1，都有可能，最后由传输延迟决定

    4.1.1.2 Basic（NOR）S-R Latch¶

    4.1.1.3 Clocked S-R Latch¶

    门控SR 锁存器是一种基于SR 锁存器的改进版本，它引入了一个额外的控制信号（通常称为使能信号或时钟信号）来控制SR 锁存器的操作。门控SR 锁存器在时钟信号的作用下，只在特定时刻才允许输入信号对锁存器进行设置或重置操作，从而实现更可控的存储功能。

    • 具有与基本 S-R 锁存器类似的时序行为，不同之处在于仅当线路 C 为高电平时才观察到 S 和 R 输入。

    • C 表示“控制”或“时钟”。

    • 作为一个 ENABLE 的功能。当$ C=0$ 时，\(Q\) 不会发生改变。当 \(C\)=1 时，上面相当于 \(\bar S\), 下面相当于 \(\bar R\), 变成一个钟控的 SR 锁存器。

    当S和R都为1，且C= 1时，是违法状态，相当于S-R Latch的\(S=0,R=0\)状态

    C S R Q(t+1)
    0 X X no change(存储状态)
    1 0 0 no change（11，存储状态）
    1 0 1 0：Clear Q
    1 1 0 1： Set Q
    1 1 1 （00）非法状态

    此处你会发现，S和R的取值组合所得结果与SR锁存器相反。

    原因在于，当C = 1时， S AND C = S，经过非门作用后，输出结果为\(\overline S, \overline R\)

    4.1.1.4 D Latch¶

    为了消除非法状态的影响，防止11的出现

    C D Q（t+1）
    0 X no change
    1 0 0 : clear Q
    1 1 1 : Set Q

    当C = 1时, Q = D

    S 和 R 输入始终保持相反，导致 0 和 0的输入也消失，也就是存储功能消失了

    但是当C = 0时，也能达到存储的结果，所以没有影响。

    锁存器的空翻现象:

    D 锁存器也存在问题。如果我门将 \(\overline Q\) 接到 D 上，就会发现 D 锁存器将不停的变化其中的数据，出现 空翻，导致我们无法确定 C 置 0 时 Q 的输出究竟是什么，而这是非常危险的。

    具体来说，当D 锁存器的输入信号在时钟信号的上升沿（或下降沿）到来时发生变化，并且在时钟信号的下一个上升沿（或下降沿）到来前，输入信号再次发生变化，就可能导致” 空翻” 现象的出现。这种情况下，D 锁存器的输出会在两个连续的时钟周期内出现两次翻转。

    解决方法：消除空翻现象，使每次触发仅使锁存器的内部状态仅改变一次.

    验证D锁存器的空翻现象

    (a) 在0-20ns，C = 0, D = 1, 得到输出结果待定

    (b) 在20-40ns，C = 1，D = 1，得到输出结果Q=1，Qbar=0

    © 在40-50ns，C = 0，D = 1，保持原状态。

    (d) 在50-60ns，D = 0，Qbar 成为输入，但是由于C=0，保持原状态。

    (e) 之后，Qbar 作为输入，C=1，开启振荡，Q 和Qbar 交替变化。

    4.1.2 Flip-Flop 触发器¶

    锁存器不适合使用在电路中：不能做到一个周期，状态只更新一次。

    只要C = 1 ， Y的值就会不断地变化

    我们期望的是：Y的值每一个时钟周期变化一次

    打断D到Q的直连通道，添加一个中间件

    4.1.2.1 Master—Slave Flip-Flop¶

    • 前面称为 master(主锁存器), 后面称为 slave(从锁存器)
    • 当 \(C=0\) 时，主锁存器不变。从锁存器处于Enable状态，S，R输入未知，Q，\(\bar Q\) 结果未知
    • 不妨设\(S= 1, Q = 0\) \(,C \(从 0 变为 1 时，主锁存器被使能，Q 改变\)Q = 1, \bar Q = 0\)，但从锁存器不变，输出结果不变。
    • \(C\) 从 1 变为 0时，主锁存器不变。从锁存器处于Enable状态，\(Q = 1, \bar Q = 0\)

    周期变长一倍 一次性采样问题(1s catching)：当 S R 均为 0 时如果有小扰动，无法复原。

    以下是一次性采样问题的解释

    S and/or R are permitted to change while C = 1

    • Suppose Q = 0 and S goes to 1 and then back to 0 with R remaining at 0

    当SR初始状态为00，S产生一个小脉冲，从0变为1，再变为0

    • The master latch sets to 1. 导致主锁存器的输出Q’变为1

    • A 1 is transferred to the slave 该结果被传输给从锁存器

    • Suppose Q = 0 and S goes to 1 and back to 0 and R goes to 1 and back to 0

    SR初始状态为00，S先从0变为1，再变为0。R再从0变为1，再变为0

    • The master latch sets and then resets

      也就是说，主锁存器先经历Set，输出结果Q‘=1，再经历Reset，输出结果变为0

    • A 0 is transferred to the slave

    • This behavior is called 1s catching

    0-25ns，C = 0， 从锁存器输出未知

    25ns-50ns，主锁存器不影响从锁存器，Q‘ = 1，输出仍然未知

    50ns-75ns，从锁存器发挥作用，Q‘=1，输出Q=1

    75ns-100ns，主锁存器发挥作用，S=0，R=1，Q’=0，但是从锁存器输出不变

    100ns-125ns，从锁存器发挥作用，由于Q’=0，输出Q=0

    125ns-150ns，主锁存器发挥作用，S=0，R=0，Q‘=1，Q’bar=1

    150ns-175ns，从锁存器发挥作用，输入是11，保持之前的状态，Q=0

    175ns-200ns，主锁存器发挥作用，中间S从0变为1，导致输出Q'变为1，之后虽然S变为0，但是S=0，R=0，得到的是保持原来的状态，也就是Q’=1

    200-225ns，从锁存器发挥作用，由于Q'=1,导致输出Q=1，但是按照原来的想法，脉冲返回原来的状态，但是Q没有按照原来的状态

    4.1.2.2 Edge-Triggered D Filp-Flop¶

    1. An edge-triggered flip-flop ignores the pulse while it is at a constant level and triggers only during a transition of the clock signal.

    边沿触发触发器会忽略处于恒定电平的脉冲，并且仅在时钟信号转换期间(边沿交界处)触发。

    1. A master-slave D flip-flop which also exhibits edge-triggered behavior can be used.

    可以使用也具有边沿触发行为的主从 D 触发器。

    1. The delay of the S-R master-slave flip-flop can be avoided since the 1s-catching behavior is not present with D replacing S and R inputs.

    由于用 D 代替 S 和 R 输入，不存在 1 捕捉行为，因此可以避免 S-R 主从触发器的延迟。

    1. D触发器发明的原因，在于解决MS FLIP-FLOP 主从触发器的一次性采样问题。D锁存器的使用直接解决了SR同时为0的情况，尽管出现脉冲，也能够恢复正常。

    Positive-Edge Triggered D Flip-Flop

    上升沿和下降沿，看的是输入到D触发器C输入端的值与CLK的关系。上升意味着，CLK从0变为1，在0阶段，实现主锁存器Q’的改变，在1阶段，实现Q‘的输出。也就是说0阶段，主锁存器发挥作用，1阶段从锁存器发挥作用。

    换句话说，从锁存器读取的Q‘是主锁存器在clk = 0 的最后一刻确定的（D触发器规避了一次性采样问题），对应的也就是边沿。

    对于上升沿，主锁存器C前面有一个非门 0 -> 1 ,进行输出

    对于下降沿，从锁存器C前面有一个非门

    (a) 当R = S = 1 时输出Q 随着输入D 的变化而变化

    (b) 在R = 0，S = 1 时，输出Q 为0

    © R = 1，S = 0 时，输出Q 为1（Direct Input，此时输出与C D 的值无关）和真值表相符，符合预期。

    (d) 上升沿触发体现在：在120ns 时，D 从0 变为1，但由于此时C 不变，所以Q 不变，直到150ns，C 从0 变为1，这时Q 才变为1

    与上面的D触发器实现的功能相同，但是门输入成本更低

    Standard symbol for storage Elements:

    Direct Input:

    • Direct R and/or S inputs that control the state of the latches within the flip-flops are used for this initialization.

    For the example flip-flop shown

    • 0 applied to \(\overline R\) resets the flip-flop to the 0 state
    • 0 applied to \(\overline S\) sets the flip-flop to the 1 state

    4.1.3 Analysis¶

    利用同步，所有的触发器都在同一时间改变

    • Current State at time (t) is stored in an array of flip-flops.
    • Next State at time (t+1) is a Boolean function of State and Inputs.
    • Outputs at time (t) are a Boolean function of State (t) and (sometimes) Inputs (t).

    例子：

    • 此处的CP也就是CLK，是时钟，时间信号不会出现在状态图（状态变迁）**，**不是外部信号输入
    • 将触发器的输出，视作电路的状态state
    • \(state ： A(t),B(t)\),n个触发器就有\(2^n\)个状态组合
    • \(Input: x(t)\)
    • \(Output: y(t)\)
    • 由于是D触发器，下一状态值由D决定(即\(Q = D\))，所以\(\(A(t+1)= D_A(t)\)\)

    4.1.3.1 State Table¶

    • Present State – the values of the state variables for each allowed state.
    • Input – the input combinations allowed.
    • Next-state – the value of the state at time (\(\(t+1\)\)) based on the present state and the input.
    • Output – the value of the output as a function of the present state and (sometimes) the input.

    行数由输入组合决定，两个触发器，两个状态，+ input = 3

    当x（t）= 0 时，Next state 均变为（0 0）

    当x（t）= 1时，Next state 历经（0，0），（0，1），（1，1），（1，0）

    每一个圆圈表示一个状态，多少个状态输入，就有\(2^n\)个圆圈

    • A directed arc from the Present State to the Next State for each state transition

    每个状态转换从当前状态到下一个状态的有向弧

    • A label on each directed arc with the Input values which causes the state transition, and A label:

    每个有向弧上的标签，其中包含导致状态转换的输入值

    输出和输入是否有关，无关标在圈里面，有关标在外面

    • On each circle with the output value produced, or
    • On each directed arc with the output value produced.
    1. **Moore** type output depends only on state(输出画在圈里面)

    输出仅取决于状态

    1. **Mealy** type output depends on state and input(输出画在有向弧上)

    输出取决于状态和输入

    斜杠左边表示外部输入，右边表示输出结果，Mealy型输出写在有向弧上

    Equivalent State Definitions

    两个状态，无论输入是什么，都会使这两个状态输出相同，次态也相同，那么这两个状态是等效状态。

    观察S2和S3，当输入为0，输出都为1，次态都为S0。当输入为1，输出都是0，次态都为S2.说明S2和S3是等效状态。

    观察S1和S2，当输入为0是，输出都为1，次态都为S0.输入为1时，输出都为0，次态都为S2，两者为等效状态。

    D触发器的个数由2个变为1个，实现了电路的简化

    Sequential Circuits or Sequential Machines are also called Finite State Machines (FSMs). Two formal models exist:

    • Moore Model
    • Outputs are a function ONLY of states
    • Usually specified on the states.
    • Mealy Model
    • Outputs are a function of inputs AND states
    • Usually specified on the state transition arcs.

    4.1.3.3 Flip-Flop Timing Parameters 触发器时序参数¶

    主从脉冲触发器（positive pulse）

    • \(t_s\) ：对于主从触发器，ts等于触发脉冲的宽度

    • \(t_w\) : clock pulse width 时钟脉冲宽度

    • \(t_s\)与C处于高电平的时间相同（这段时间SR不能发生改变），否则会产生一次性采样问题

    下边沿触发器（negative edge），C从高电平变为低电平时触发。

    • \(t_s\) : setup time. 在C下降之前要先准备好。等于通常远小于触发脉冲宽度的时间间隔

    • \(t_h\) : hold time. 触发之后还要保持一段时间，通常等于0

    • \(t_{px}\) : propagation delay. 传输延迟

    Measured from clock edge that triggers the output change to the output change.

    根据定义, \(t_s\)指在采样前触发器需要的准备时间。对于主从触发器，在信号处于上升沿的时候 S 部分的输入是不能改变的，否则可能有一次性采样问题，因此在信号处于上升沿都是触发器的准备时间。这也是为什么现在我们普遍使用边缘触发器的原因。

    对于 D 触发器而言，不要卡在时钟边沿改变 D 的输入，而是在时钟改变之前就进行，这样 D 触发器才能有足够的准备时间。

    4.1.3.4 Circuit and System Level Timing¶

    • \(t_p\) : clock period

    • $t_{pd,COMB} : $ total delay of combinational logic along the path from flip-flop output to flip-flop input

    从触发器输出到触发器输入路径上组合逻辑的总延迟

    • \(t_{slack}\) : extra time in the clock period in addition to the sum of the delays and setup time on a path. 松弛时间（休息）必须大于等于 0.才能正确运行

    对于Edge-triggered ， 先是(\(t_{pd，FF}\)\)触发器传输延迟 ，再(\(t_{pd,COMB}\)\),组合逻辑电路传输延迟，(\(t_s\)\)是set time，准备时间

    对于Pulse- triggered ，(\(t_s\)\)占据一整个低电平的时间

    (\(t_{slack}\)\) 必须大于等于0，才能够正常工作。

    边沿触发的(\(t_s\)\)较小，脉冲触发的(\(t_s\)\)较大，为了尽可能缩小(\(t_p\)\),尽可能减小组合逻辑电路的传输延迟

    Time equation: \(t_p≥max(t_{pd,FF}+t_{pd,COMB}+t_s)\) for all paths from flip-flop output to flip-flop input.

    Compare the allowable combinational delay for a specific circuit:

    • Using edge-triggered flip-flops;
    • Using master-slave flip-flops.

    给定一些parameter ：

    • clock frequency = 250 MHz
    • \(t_{pd,FF}(max) = 1.0ns\)
    • $t_s(max) = 0.3ns $for edge-triggered flip-flops
    • \(t_s(max) = t_{wH} = 2.0ns\) for master-slave flip-flops

    利用Timing Equations $$ t_p = \dfrac{1}{clock frequency} = 4.0ns\ \ 对于Edge-triggered :\ 4.0 >= t_{pd,FF} + t_{pd,COMB} + t_s = 1.0ns + t_{pd,COMB} + 0.3ns\ 得到t_{pd,CMOB} <= 2.7ns\ \ 对于Master-slave:\ 4.0 >= t_{pd,FF} + t_{pd,COMB} + t_s = 1.0ns + t_{pd,COMB} + 2.0ns\ 得到得到t_{pd,CMOB} <= 1.0ns\ $$
