    Programmable Implementation Technologies

    5.1 Why Programmable Logic？¶

    • Facts:

    • It is most economical to produce an IC in large volumes经济实惠

    • Many designs required only small volumes of ICs

    • Need an IC that can be:

    • Produced in large volumes大批量生产

    • Handle many designs required in small volumes处理小批量所需的多种设计

    • A programmable logic part can be:

    • made in large volumes

    • programmed to implement large numbers of different low-volume designs 编程以实现大量不同的小批量设计

    • Many programmable logic devices are field-programmable, i. e., can be programmed outside of the manufacturing environment

    许多可编程逻辑器件是现场可编程的，即。即，可以在制造环境之外进行编程

    • Most programmable logic devices are erasable and reprogrammable.

    大多数可编程逻辑器件都是可擦除和可重新编程的。

    • Allows “updating” a device or correction of errors

      允许“更新”设备或纠正错误

    • Allows reuse the device for a different design - the ultimate in re-usability!

    • Ideal for course laboratories

    • Programmable logic devices can be used to prototype design that will be implemented for sale in regular ICs.

    • Complete Intel Pentium designs were actually prototyped with specialized systems based on large numbers of VLSI programmable devices!

    5.2 Programming Technologies¶

    5.2.1 Classification¶

    Programming Technologies are used to

    • Control connections

    把两个脚连接起来，常见的连接方法有：

    • Mask programming 一次性编程，工厂里芯片制造时使用，厂家在生产时编程到芯片内
    • Fuse(熔丝) 在芯片设计时，在很多芯片引脚连接之间有 fuse, 设计电路时不需要保留连接的用高电压产生高电流，熔断 fuse.
    • Antifuse 最开始断开，击穿后电路连接好

    • Single-bit storage element 存储一个二进制位，这个位的输出控制一个晶体管，进而控制两个引脚连接/断开

    • Build lookup tables(LUT) 如 16 位寄存器（存真值表的值） +16-1 Mux 可构成四输入之内的逻辑模块，只需要将 16位寄存器填上不同的值就可以实现不同的逻辑函数

    • Control transistor switching

    用晶体管控制开关，类似连接控制

    浮动栅极

    • Stored charge on a floating transistor gate
      • Erasable 可擦除
      • Electrically erasable
      • Flash (as in Flash Memory)
    • Storage elements

    5.2.2 Character¶

    • Permanent - Cannot be erased and reprogrammed

    永恒性： 无法擦除和重新编程

    • Mask programming 一次性编程顾名思义
    • Fuse 熔丝

    • Antifuse 反熔丝都只能实现一次

    • Reprogrammable

    • Volatile(易失的) - Programming lost if chip power lost

      • Single-bit storage element
    • Non-Volatile
      • Erasable
      • Electrically erasable
      • Flash (as in Flash Memory)

    5.3 Configuration 配置¶

    • Read only Memory(ROM) ——a fixed array of AND gates and a programmable array of OR gates

    只读存储器 (ROM) - 固定的与门阵列和可编程的或门阵列

    • Programmable Array Logic (PAL) - a programmable array of AND gates feeding a fixed array of OR gates. 可编程阵列逻辑 (PAL) - 一个可编程的 AND 门阵列，为固定的 OR 门阵列提供数据。

    • Programmable Logic Array (PLA) - a programmable array of AND gates feeding a programmable array of OR gates. 可编程逻辑阵列 (PLA) - 为可编程或门阵列提供信号的可编程与门阵列。

    AND门阵列和OR门阵列都是可编程的

    • Complex Programmable Logic Device (CPLD) /Field- Programmable Gate Array (FPGA) - complex enough to be called “architectures” - See VLSI Programmable Logic Devices reading supplement 复杂可编程逻辑器件 (CPLD)/现场可编程门阵列 (FPGA) - 复杂到足以被称为“架构” - 请参阅 VLSI 可编程逻辑器件阅读补充

    5.3.1 Read Only Memory¶

    Read Only Memories (ROM) or Programmable Read Only Memories (PROM) have: N input lines, M output lines, and \(2^N\) decoded minterms.

    只读存储器（ROM）或可编程只读存储器（PROM）具有： N 个输入线， M 个输出线，以及 \(2^N\) 解码（decoder）的最小项

    利用译码器decoder + or门，实现任意的逻辑函数。

    可以利用decoder生成Minterm最小项，再用OR门连接起来

    • Fixed AND array with \(2^N\) outputs implementing all N-literal minterms.

    • Programmable OR Array with M outputs lines to form up to \(2^N\) sum of minterm expressions.

    具有 M 条输出线的可编程 OR 数组，可形成最多 M 个最小项表达式之和。

    二维阵列：AND 阵列产生 \(2^N\) minterms, OR 阵列有 M 个输出，每个输出对应最多 \(2^N\) minterms.

    此处是一个\(8 \times 4 \ ROM (N \ = \ 3 \ input \ lines, M \ = \ 4 \ output \ lines)\)

    固定的AND门数量，具有3条输入线，实现8个最小项输出

    注意高低位的连接

    5.3.2 Programming Array Logic（PAL）¶

    The PAL is the opposite of the ROM, having a programmable set of ANDs combined with fixed ORs.

    PAL 与 ROM 相反，具有一组可编程的 AND 与固定的 OR 组合。

    • Disadvantage
      ROM guaranteed to implement any M functions of N inputs. PAL may have too few inputs to the OR gates. ROM保证实现N个输入的任意M个功能。 PAL 的“或”门输入可能太少。（每一个OR门只能接受指定数量的输入）
    • Advantages
    • For given internal complexity, a PAL can have larger N and M 对于给定的内部复杂性，PAL 可以具有更大的 N 和 M
    • Some PALs have outputs that can be complemented, adding POS functions 一些 PAL 具有可补充的输出（下图的线8，9），添加 POS 功能
    • No multilevel circuit implementations in ROM (without external connections from output to input). PAL has outputs from OR terms as internal inputs to all AND terms, making implementation of multi-level circuits easier. ROM 中没有多级电路实现（没有从输出到输入的外部连接）。 PAL 将 OR 项的输出作为所有 AND 项的内部输入，使多级电路的实现更加容易。

    AND门可编程（也就是可以控制输入的个数）

    OR门固定（也就是输入个数确定，不足的话补充1，过多的话将OR项的输出重新作为AND项的内部输入）

    每一个AND门，可以接任意个输入，每一个OR门，只能接3个输入。

    对于多输入的OR门需求，可以将F1重新接入到线8，9，通过F2，F3，F4进行输出

    上图中， \(F1 = \overline A \ \overline B + \overline C\)

    \(F2 = \overline A B \overline C + AC + A \overline B\)

    \(F3 = AD + BD + \overline A \ \overline B + \overline C\)

    \(F4 = AB + CD + AC + BC(\overline{F1})\)

    • 1 表示 接入原项
    • 0 表示 接入取反项
    • ——表示不接入

    观察到，W重新作为AND项的输入，为了得到F1

    5.3.3 Programming Logic Array¶

    Compared to a ROM and a PAL, a PLA is the most flexible having a programmable set of ANDs combined with a programmable set of ORs. 与 ROM 和 PAL 相比，PLA 最为灵活，具有一组可编程的 AND 与一组可编程的 OR。

    • Advantages
    • A PLA can have large N and M permitting implementation of equations that are impractical for a ROM (because of the number of inputs, N, required) PLA 可以具有较大的 N 和 M，从而允许实现对于 ROM 来说不切实际的方程（因为需要输入的数量 N）
    • A PLA has all of its product terms connectable to all outputs, overcoming the problem of the limited inputs to the PAL Ors PLA 的所有乘积项都可连接到所有输出，克服了 PAL Ors 输入有限的问题
    • Some PLAs have outputs that can be complemented, adding POS functions 一些 PLA 具有可补充的输出，添加 POS 功能
    • Disadvantages
    • Often, the product term count limits the application of a PLA. 通常，产品术语计数限制了 PLA 的应用。
    • Two-level multiple-output optimization is required to reduce the number of product terms in an implementation, helping to fit it into a PLA. 需要进行两级多输出优化来减少实现中的乘积项数量，从而有助于将其融入 PLA 中。
    • Multi-level circuit capability available in PAL not available in PLA. PLA requires external connections to do multi-level circuits. PAL 中提供的多级电路功能在 PLA 中不可用。 PLA需要外部连接来实现多级电路。

    以此为例：你会发现，前面的AND门可以控制输入的个数，同时后面的OR门也能控制AND输出项作为输入的个数。也就是同时实现AND 和 OR同时可编程

    此处最后的异或门，能够缓减AND门数量不够的缺点。假如没有异或门，需要实现AB，BC，AC，A\(\overline B\), \(\overline AB\),但是AND门的数量不够

    \(G1 = AB + BC + AC\)

    \(G2 = AB + \overline A \ \overline B\)

    \(F1 = 0 \oplus G1 = G1 = AB + BC + AC\)

    \(F2 = 1 \oplus G2 = \overline {G2} = (A + B)(\overline A + \overline B) = A\overline B + \overline AB\)

    此处对F1进行求反，\(\overline{F1}\)需要四项，F2所需的3项都在\(\overline{F1}\)中出现，符合限定4个OR门的要求

    此处F1求反，和F2求反，在此处均适用于四个与门的PLA

    通过求反，实现共用AND与项
